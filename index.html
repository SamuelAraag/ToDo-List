Token do GitHub Necessário

Para buscar e salvar suas tarefas, por favor insira seu Personal Access Token do GitHub. ?

Como conseguir o token?

  1. Vá em Settings > Developer settings > Personal access tokens > Tokens (classic) no GitHub.
  2. Clique em Generate new token.
  3. Dê um nome, defina a expiração e selecione o escopo repo.
  4. Clique em Generate token e copie o valor.
